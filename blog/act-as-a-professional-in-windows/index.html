1- Chocolatey - A Software Package Manager

I believe you've heard of apt (advanced package tool). It's an Ubuntu package manager for downloading, installing and managing softwares.

There is a similar package manager for windows called: Chocolatey or for short: choco. to install, you can find the full and the newest instructions on their website: here.

Or do the following steps:

  1. Open up Powershell as administrator

    • Search for powershell in windows search-bar
    • Enter "Windows PowerShell" program using this combination: CTRL + Shift + Enter
    • Click Yes when the pop-up opens up

  2. Run one of the following commands to set some flags

    • Set-ExecutionPolicy AllSigned
    OR
    • Set-ExecutionPolicy Bypass -Scope Process

  3. Now run the command below to install chocolatey:

    Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))
  4. Wait a few seconds for the command to complete. if you don't see any errors, you are ready to use Chocolatey!
  5. Type choco -? to see the options.
    You can also see the documentations on their website here: Getting Started


Do not ever forget, you should run command-line (cmd) or powershell as administrator each time you wanna install or remove any packages.

Just for giving you an example, let's install python 3.7 using chocolatey. open up cmd as administrator. then type the command below:

choco install python --version=3.7.2

For more details and to see all the available packages, please visit their website!

2- Git-Bash - A Bash Emulator

You can see the download page here: Gitbash

But You've already learnt how to install via chocolatey package manager. so why not to use it?!

Open up a command-line (cmd or powershell) as administrator and copy the command below to install the latest version:

choco install git


After that, you'll be able to find the installed software in windows search-bar. search for: Git Bash. You might not be able to find it, if you don't type the complete words!
You can pin it to taskbar afterwards, so that you won't need to search that again next time.

Most important commands of linux shell has been inserted into gitbash 😍 (including the git):

  • ls
  • git
  • vim
  • ssh
  • grep
  • file

...

You can of course, pipe commands. it also supports a virtual filesystem-tree that starts from the root ( / ), just like unix-based systems.

Gitbash consists two types of application:

  • Git GUI
  • Git Bash

3- VSCode - A Rich Editor

VScode or Visual Studio Code is an omnipotent and open-source editor for all types of programming. You can easily find the best available extensions for your field of work.

There is two types of vscode available for download. both are open-source and available for all:

  • VSCode
  • VSCode Insiders

VSCode-Insiders is a prerelease-version of common vscode. according to an answer in Quora:

Visual Studio Code Insiders is just a beta version of visual studio code. It includes the latest features, but those features may or may not be stable.

You know how to install it. Don't You?!

choco install vscode


OR for insiders:

choco install vscode-insiders

You decide which one you wanna be installed. I personally prefer vscode-insiders because it auto-updates minor updates and no big deal for me!

After installing, you definitely should change the integrated terminal of vscode from the default shell to GitBash:

  1. Open VSCode 🙄

  2. Do this internal command:

    • open up vscode command-area using this combination: Ctrl + Shift + P
    • Thereafter the command-area got opened, type for: `Default Shell`
    • Enter the option: "Terminal: Select Default Shell"
    • Select the Git Bash item
  3. Restart VSCode 😕


OR You can directly go to the setting page and search for default shell/terminal.

NOTE: If you couldn't find the `Git Bash` option in the selection panel, you may have not installed the git-bash or you haven't added its path to the PATH environment variables!

4- ADVOR - Advanced Onion Router

You gotta heard about the TOR Project. you can access the tor-proxy in windows via a GUI window. according to their description:

Advanced Onion Router (AdvOR) is a portable client for the OR network and is intended to be an improved alternative for Tor+Vidalia+Privoxy bundle for Windows users

How does ADVOR work?

It creates a proxy listener on port 9050 for both SOCKS5 and HTTP on your localhost.

How does ADVOR look?

It's an easy-to-use GUI-based software for using TOR without having any more concerns about setting up a proxy by self hands.

How to install ADVOR?

Don't ask, Just like always 😑

choco install advor

In case you're rookie in proxy usage

You can find a detailed description about proxies in wikipedia here: Proxy Server

Using proxies might have multiple reasons, but a common case is when your ISP blocks accessing some contents on Internet. your first choice can be using VPN or Proxy servers.

TOR Socket Address

  • Hostname: 127.0.0.1 or localhost
  • Port: 9050

ADVOR is using TOR so you won't need to add any credentials (Username/Passwords).

5- Proxychains - Running Through Proxy Server

As Vickie Li explains in this post:

ProxyChains is a tool that forces any TCP connection made by any given application to go through proxies like TOR or any other SOCKS4, SOCKS5 or HTTP proxies.
It is an open-source project for GNU/Linux systems. Essentially, you can use ProxyChains to run any program through a proxy server.

Remember you have already installed TOR through installing ADVOR in the previous section. so you won't need to install that again, whenever you connect ADVOR, the TOR service will be started!

What you need to install, is ProxyChains.
You might ask yourself you've already got the ADVOR software, "so why do I need proxychains?".
I gotta say, ADVOR just provides a proxy-server listener and you cannot run a cli-based application through the proxy if it doesn't support an option for proxy.

Let me explain a real case to you 🙉

Suppose that you want to push to a remote server after committing a repository via git. but Unfortunately, GitLab has blocked your country and whenever you push that, you'll get a 403 Forbidden Error ☹

So, one idea could be using proxychains, doing a command like:
proxychains git push origin master.

Yup, proxychains passes the requests through a proxy (by default the TOR service).

Fortunately, you can have proxychains in Windows. there is an open-source repo in github: proxychains-windows.

How to install ProxyChains?

  1. Open this download link. it is the releases-page of the github repo I talked about previously.
  2. Download the latest proper release application that meets your OS requirements. (e.g. proxychains0.6.8win32_x64.zip)
  3. Afterwards, extract and put it into any places you like. (e.g. C:\Program Files\ProxyChains).

  4. Go to the directory that you just extracted and placed it somewhere.
    Find the proper Executable Application. (e.g. proxychainshelperwin32_x64.exe)

    You need to rename it to proxychains (all letters lowercase).
    This will enable you to be able to run ProxyChains easily by using this command in the terminal/cmd: proxychains

  5. Find and open up the Environment Variables panel by searching in windows search-bar.
    You gotta add the path of the directory you just extracted into the PATH.

    If you got troubles adding to the %PATH%, just read this question on StackExchange here.

  6. After that, you can open a new cli (cmd, gitbash, ...) and run the command. do this for the help: proxychains -?


Happy Hacking!

For more tutorials about this topic, read this article: ProxyChains Tutorial.

6- Postman - A REST API Client

Send requests, inspect responses, and easily debug REST APIs.
Use Postman as a REST client to create and execute queries. (Postman also works with SOAP and GraphQL.)

You can use this application to send or test HTTP Requests as easy as possible. I believe every developers need to have sth like this. it is always handy and useful for many purposes like: Front-End Development, Back-End Development, Penetration Testing and etc.

choco install postman
